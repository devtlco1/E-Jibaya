# سكريبت استيراد السجلات من ملفات PDF

هذا السكريبت يستخرج أرقام الحسابات وأرقام المقاييس من ملفات PDF ويضيفها كسجلات فارغة إلى قاعدة البيانات.

## المتطلبات

1. تثبيت المكتبات المطلوبة:
```bash
npm install
```

2. التأكد من وجود ملف `.env` يحتوي على:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## الاستخدام

1. ضع ملفات PDF في مجلد `DATA/`:
   - `DATA/r80_rej.pdf`
   - `DATA/r80.pdf`

2. قم بتشغيل السكريبت:
```bash
npm run import-pdf
```

## ما يفعله السكريبت

1. يقرأ ملفات PDF من مجلد `DATA/`
2. يستخرج أرقام الحسابات (12 رقم) وأرقام المقاييس (5-8 أرقام)
3. يتحقق من عدم وجود السجلات مسبقاً في قاعدة البيانات
4. يضيف السجلات الفارغة الجديدة (تحتوي فقط على رقم الحساب ورقم المقياس)
5. يعرض ملخص العملية (عدد السجلات المضافة، المكررة، والأخطاء)

## ملاحظات

- السجلات المضافة ستكون بحالة `pending` (معلق)
- السجلات ستكون فارغة (بدون بيانات أخرى)
- السكريبت يتجنب إضافة السجلات المكررة
- يتم إضافة تأخير صغير كل 50 سجل لتجنب الضغط على قاعدة البيانات
